Operador ternário

É uma estrutura de decisão, semelhantr ao if-else. A diferença é que ele possui uma sitax bem mais enxuta, fazendo com que o nosso código fique menos verboso. Por outro lado ele é um pouco mais limitado.

Sintax:

var resultado = <condição> ? <verdadeiro> : <falso>

Exemplo baseado na última aula:

var resultado = (nota >= media && faltas <= faltas_maximas) ? 'Aprovado' : 'Reprovado'
