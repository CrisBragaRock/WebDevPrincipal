Comando If / Else

O comando if-else, consiste em uma estrutura de controle. Ele nos possibilita, com base em uma determinada condição, definir qual fluxo que a aplicação vai tomar. É atravéz dele que podemos fazer diversas verificações para criar fluxo dentro das nossas aplicações que atendam a todos os tipos de possibilidades, pelo menos, todas aquelas que sejam possíveis de serem previstas.

É muito comum criarmos aplicações que não controlam determinados fluxos, e isso acontece por inumeros motivos, desde um levantamento de requisito superficial, até mesmo por distração ou negligência. Nesses casos teremos potenciais fontes de bug dentro dessas aplicações, então, quanto mais controlarmos essas situações, mais confiáveis serão as nossas aplicações.

A sintax do comando é bem simples: 
if (condição) {
    -> trecho de código que será executado
} else {
    -> trecho de código que será executado
}


------------------------------------------

if (condição) {
    -> trecho de código que será executado
} else if (condição) {
    -> trecho de código que será executado
} else {
    -> trecho de código que será executado
}
