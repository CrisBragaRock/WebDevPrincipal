EVENTOS (MOUSE)

Eventos:

  • onclick: acionado quando o elemento é clicado.
  • ondblclick: acionado quando o elemento é clicado 2 vezes.
  • onmouseup: acionado quando o click sobre um determinado elemento é liberado.
  • onmouseover: acionado quando o cursor do mouse sobrepõe o elemento.
  • onmouseout: acionado quando o cursor do mouse sai da região da página ocupada pelo elemento.
