DIFERANÇAS ENTRE O ATRIBUTO LENGHT DE VARIÁVEIS DO TIPO STRING E ARRAY

O atributo lenght está disponível para objetos do tipo array e também para variáveis do tipo string. Se você se lembra bem o atributo para strings retorna a quantidade de caracteres contidos dentro de uma string. Já em objetos do tipo array, o atributo lenght retorna a quantidade de elementos dentro do array. Só que no caso de arrays ele pode conter um valor inesperado, isso porque ele conterá a quantidade de elementos dentro do array desde que esses elementos sejam indexados por valores sequências sem intervalo e que iniciem em zero.
