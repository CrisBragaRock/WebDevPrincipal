Introdução a Funções

As funções tem o objetivo de encapsular um determinado bloco de códigos que possui um objetivo bem definido, e, atravéz das chamadas das funções, nós podemos recuperar essas instruções para reulizar elas em diversas partes das nossas aplicações. A declaração de uma função é bastante simples.

Sintax:

                    
    function calcularAreaTerreno(largura, comprimento) {
        var area = largura * comprimento
        
        return area
    }

Geralmete é indicado usar duas palavras para a atribuição dos nomes das funções, sendo a primeira um verbo no infinitivo e a segunda um substantivo iniciado com a primeira letra em maiúsculo.

As funções podem ser de dois tipos diferentes. São eles:

  • Void: Quando chamada ela irá apenas processar alguma lógica.
  • Return: Como no exemplo acima. Ela processa a lógica exigida e nos tras algum valor de retorno, ou até mesmo de uma variável para o script que efetuou a chamada da função.
