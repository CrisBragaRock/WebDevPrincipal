Operadores de comparação (condicionais)

Também conhecidos como operadores relacionais, servem para formar as expressões condicionais para o comando if.

Operadores

Igual (==) - Verifica se os valores comapados são iguais.

Idêntico (===) - Verifica se os valores comapados são iguais e do mesmo tipo.

Diferente (!=) - Verifica se os valores comapados são diferentes.

Não idêntico (!==) - Verifica se os valores comapados são diferentes e de tipos diferentes.

Menor (<) - Verifica se o valor da esquerda é menor que o da direita.

Maior (>) - Verifica se o valor da esquerda é maior que o da direita.

Menor igual (<=) - Verifica se o valor da esquerda é menor ou igual ao da direita.

Maior igual (>=) - Verifica se o valor da esquerda é maior ou igual ao da direita.
