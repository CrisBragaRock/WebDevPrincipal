EVENTOS(TECLADO)

Eventos:

  • onkeydown:

    • Acionado quando uma tecla é pressionada

  • onkeypress:

    • Acionado enquanto a tecla continua pressionada(funciona somente para caracteres e nao para teclas especiais como o CTRL)

  • onkeyup:

    • Acionado quando uma tecla é liberada
