Switch

Assim como o comando if, o comando Switch é um condicional, ou seja, ele visa com base em uma determiada condição, seguir um determinado fluxo dentro da aplicação. Na verdade o comado Switch é realmente igual ao if, porém é escrito de uma maneira diferente, e nesse ponto ele nos auxilia no entendimento do código, já que a forma de escrita do comando Switch é mais intuitiva.

Ele é restrito porque só conseguimos fazer operações de comparação de indênticos, mas, em situações, por exemplo, de um menu, em que nesse menu é defindo o fluxo da aplicação, o comando Switch é bastante interessante porque ele organiza o nosso código de uma maneira muito mais fácil de entender.

Exemplo:


var opcao = 2

switch (opcao) {
    
    case 1:
    //trecho de código que será executado
    break
    
    
    case 2:
    //trecho de código que será executado
    break
    
    
    default:
    //trecho de código que será executado
    break
    
}

Entre os parenteses, nós passamos a opção, o valor que será utilizado como parâmetro de comparação nos cases. Essa opção, pode ser uma variável ou até mesmo um valor pré-definido.
